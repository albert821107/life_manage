📌 取得 LINE Notify Token

  1. 前往 LINE Notify
  2. 登入 LINE 帳號
  3. 產生個人存取權杖
  4. 複製 Token 填入 .env 的 LINE_NOTIFY_TOKEN

在 .env 設定：
DAILY_NOTIFY_ENABLED=true
DAILY_NOTIFY_CRON=0 8 * * *
