ChatGPT (OpenAI)
GPT-4o、GPT-4o-mini 等模型
未設定
1. 前往 OpenAI Platform
開啟 platform.openai.com 並登入帳號
2. 建立 API Key
左側選單 → API Keys → 「+ Create new secret key」→ 複製後妥善保管（只顯示一次）
3. 加值帳戶
Billing → Add payment method → 儲值後方可使用（最低約 $5 USD）

GitHub Copilot
程式碼 AI 助理，整合 VSCode / JetBrains 等 IDE
ℹ️ GitHub Copilot 屬於訂閱制，透過 IDE 外掛使用，不需要 API Key，無法透過本系統直接呼叫。
1. 訂閱 GitHub Copilot
前往 github.com/features/copilot，個人方案 $10/月，學生可申請免費
2. 安裝 VSCode 擴充
在 VSCode 搜尋 GitHub Copilot 並安裝 → 使用 GitHub 帳號登入授權
3. 使用 Copilot Chat
安裝 GitHub Copilot Chat 擴充 → 側邊欄即可開啟 AI 對話
4. 適用場景
程式碼補全、解釋程式碼、生成函式、Debug、撰寫測試 — 專注在開發輔助，與 ChatGPT/Claude 的通用 AI 定位不同

尚未設定 Bybit API

            
依照以下步驟取得 API Key 並填入 .env

          


          
          

            

              
1

              

                
登入 Bybit 帳戶

                
前往 Bybit 官網並登入你的帳戶（需完成 KYC 驗證）

              

            

            

              
2

              

                
進入 API 管理頁面

                
右上角頭像 → 我的帳戶 → API 管理（或直接搜尋「API Management」）

              

            

            

              
3

              

                
建立新的 API Key

                
點擊「建立新金鑰」→ 選擇 系統生成的 API Keys

                權限勾選：✅ 讀取（必要）　❌ 提款（不要勾）

                IP 限制：可填入本機 IP 提高安全性，或留空

              

            

            

              
4

              

                
複製 API Key 與 Secret

                
建立後頁面會顯示 API Key 和 API Secret（Secret 只顯示一次，請立即複製）

              

            

            

              
5

              

                
填入 .env 檔案

                
開啟專案根目錄的 .env 檔案，填入以下設定後重啟伺服器：

                

                  BYBIT_API_KEY=貼上你的 API Key

                  BYBIT_API_SECRET=貼上你的 Secret

                  BYBIT_TESTNET=false
                

              

            

          


          

            ⚠️ 本系統僅使用「讀取」權限查詢帳戶資訊，不會執行任何交易或提款操作
