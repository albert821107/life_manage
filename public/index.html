🟢 LINE Notify

📌 取得 LINE Notify Token

  1. 前往 LINE Notify
  2. 登入 LINE 帳號並產生個人存取權杖
  3. 填入 .env 的 LINE_NOTIFY_TOKEN

📌 每日自動摘要
DAILY_NOTIFY_ENABLED=true

✈️ Telegram Bot

📌 設定 Telegram Bot

  1. 對 @BotFather 發送 /newbot 建立 Bot
  2. 取得 Bot Token 填入 TELEGRAM_BOT_TOKEN
  3. 對 Bot 發一則訊息後，開啟
    https://api.telegram.org/bot<TOKEN>/getUpdates
    取得 chat.id
  4. 填入 TELEGRAM_CHAT_ID
