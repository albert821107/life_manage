# ROADMAP — 個人管家

## ✅ v1.0.0 — 核心功能（當前版本）

### 💰 記帳
- [x] 收入 / 支出新增與刪除
- [x] 月份篩選與結餘計算
- [x] 類別分類（餐飲、交通、薪資…）
- [x] 月度統計 API

### ✅ 任務
- [x] 新增 / 編輯 / 刪除任務
- [x] 三欄 Kanban 看板（待辦 / 進行中 / 完成）
- [x] 優先級（高 / 中 / 低）標示
- [x] 到期日逾期警示

### 💪 健身
- [x] 運動類型記錄（跑步、重訓、游泳…）
- [x] 時長與熱量追蹤
- [x] 本週 / 本月統計

### 📈 投資
- [x] 持倉管理（股票、ETF、加密貨幣、債券）
- [x] 買入 / 賣出交易記錄
- [x] 自動計算平均成本與損益
- [x] 手動更新現價

### 🤖 AI 助理
- [x] OpenAI API 整合（gpt-4o-mini）
- [x] 模擬模式（無需 API Key 也可使用）
- [x] 上下文感知（自動帶入財務、任務、健身資料）
- [x] 對話歷史持久化

### 🔔 LINE 通知
- [x] LINE Notify Token 設定
- [x] 自訂訊息推播
- [x] 每日摘要一鍵發送
- [x] 排程自動推播（node-cron）
- [x] 通知記錄歷史

### 🏗 架構
- [x] Node.js + Express 後端
- [x] SQLite 本地資料庫（零設定）
- [x] Socket.io 即時更新
- [x] PM2 部署設定
- [x] 深色側邊欄 + 淺色主內容 SPA UI
